1. 树的高度

Leetcode : 104. Maximum Depth of Binary Tree (Easy)

树是一种递归的定义：一棵树要么是空树，要么有两个指针，每个指针指向一棵树。因此很多树相关的问题都能用递归的方式求解。

这题要求解树的高度，可以使用递归的思路求解：一颗树如果是空树，则高度为 0，否则它的高度是两个子树高度最大值再加 1，加 1 是因为根节点的高度为 1。可以把两颗子树当做新的树，因此求解两颗子树的高度可以继续调用求解方法。

public int maxDepth(TreeNode root) {
    if (root == null) return 0;
    return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
}

2. 平衡树

Leetcode : 110. Balanced Binary Tree (Easy)

所谓的平衡树，是指每个节点左右子树的高度差不大于 1。上图中左边的树满足要求，但是右边的树不满足，节点 0 左子树的高度为 3，右子树高度为 1，高度差为 2。

根据定义，我们可以依次求解出左右子树的高度，判断它们只差的绝对值是否大于 1。如果树中有任意一个节点不满足要求，那么结果就为 false。

private boolean result = true;

public boolean isBalanced(TreeNode root) {
    maxDepth(root);
    return result;
}

public int maxDepth(TreeNode root) {
    if (root == null) return 0;
    int l = maxDepth(root.left);
    int r = maxDepth(root.right);
    if (Math.abs(l - r) > 1) result = false;
    return 1 + Math.max(l, r);
}

3. 归并两棵树

  1. Merge Two Binary Trees (Easy)

题目要求将两颗树归并为一颗，如果在某个节点处，两棵树在该节点上都不为空，那么将两个节点的值加起来，否则只取有值的那颗树的节点的值。

public TreeNode mergeTrees(TreeNode t1, TreeNode t2) {
    if (t1 == null && t2 == null) return null;
    if (t1 == null) return t2;
    if (t2 == null) return t1;
    TreeNode root = new TreeNode(t1.val + t2.val);
    root.left = mergeTrees(t1.left, t2.left);
    root.right = mergeTrees(t1.right, t2.right);
    return root;
}
